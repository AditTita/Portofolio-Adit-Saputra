Tentang Saya

Adit Saputra

Saya adalah mahasiswa Program Studi Teknik Informatika di Institut Teknologi Sumatera (ITERA). Saat ini saya sedang berada dalam tahap awal perjalanan akademik dan sedang fokus mempelajari dasar-dasar dalam bidang teknologi informasi, terutama dalam pengembangan web dan pemrograman.

Sebagai mahasiswa yang baru memulai, saya telah mengenal beberapa bahasa dan alat dasar seperti C++ untuk memahami logika pemrograman, HTML dan CSS untuk membuat serta mendesain halaman web, serta Figma sebagai alat bantu dalam merancang tampilan antarmuka pengguna (UI). Selain itu, saya juga mulai belajar menggunakan GitHub sebagai media untuk menyimpan dan mengelola proyek secara online.

Saya memiliki ketertarikan besar terhadap dunia teknologi, terutama dalam pengembangan aplikasi web dan desain antarmuka yang menarik serta fungsional. Saya percaya bahwa dengan konsistensi belajar dan kemauan untuk terus berkembang, saya dapat menjadi bagian dari generasi muda yang mampu memberikan kontribusi positif di bidang teknologi informasi.

Pengalaman

Saya belum memiliki pengalaman profesional di bidang teknologi informasi, namun saya aktif belajar dan berlatih secara mandiri. Saya telah mencoba membuat halaman web sederhana menggunakan HTML dan CSS, mempelajari dasar pemrograman dengan C++, serta merancang tampilan antarmuka menggunakan Figma.
